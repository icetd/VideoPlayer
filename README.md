## Build
### UNIX

```
make sure you have FFmpeg
sudo apt install or build from source

mkdir build && cd build
camke .. && make
```

### WIN
```
creat VS poject with cmake GUI.
View the .gitignore and the CMakeLists.txt
to download the corresponding library
```
## Use
```
cd build
./VideoPlayer

input your url for rtsp rtmp or filePath
```
